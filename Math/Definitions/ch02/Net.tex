\begin{df}[\Net]
    A \Net is a \Function 
    mapping from a directed set $(A, \leq)$
    into another set $X$. 
    If $f:A \to X$ is a \Net
    such that for $\alpha \in A$ we have
    $f(\alpha) = x_{\alpha}$, then we may 
    use the notation
    $\{x_\alpha\}_{\alpha \in A} \subset X$. 
\end{df}
