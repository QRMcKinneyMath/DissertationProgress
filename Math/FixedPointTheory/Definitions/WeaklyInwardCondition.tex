\begin{df}[\WeaklyInwardCondition]
\label{def:InwardCondition}
\rm
Let $X$ be a \TVS. 
Let $K \subset X$. 
Let $T:K \to X$. 
We say that $T$ satisfies the \WeaklyInwardCondition if 
for each $x \in K$, $Tx \in \overline{I_K(x)}$. 
We say that $T$ satisfies the \InwardCondition if for each $x \in K$, 
$Tx \in I_K(x)$. 
Note that an operator can be said to possess the \InwardCondition even if the space $X$ lacks a 
\Topology.
\end{df}
